# Live2D Model Assets

Place your Cubism 4 Live2D model files in this directory.

The renderer looks for:

```text
src/assets/model3.json
```

Keep the referenced `.moc3`, texture, physics, pose, motion, and expression files beside the model settings file, preserving the relative paths generated by Cubism.

If no model is present, the overlay shows a lightweight placeholder avatar so the WebSocket bridge, speech bubble, and voice indicator can still be tested.
